# Example Workflow

This is a template showing how to create workflows for common tasks.

## Steps

1. **First Step**: Describe what to do
   - Additional details or context
   - Example commands if needed

2. **Second Step**: Next action
   ```bash
   # Example command
   echo "Hello World"
   ```

3. **Third Step**: Final action
   - Verification steps
   - Expected outcomes

## Notes

- Use `// turbo` annotation above a step to auto-run that specific step
- Use `// turbo-all` annotation to auto-run all command steps in the workflow
- Workflows can be triggered with `/workflow-name` slash commands
